## 关于组件库
基于物流现有业务，抽离出一系列关于React的前端组件。主要应用于小度驿站、小度众包等产品中。

## 安装
```bash
npm install fivesix --save
```

## 使用方法
```jsx
import { DatePicker } from 'fivesix/lib/basic/TenDaysSelect';
ReactDOM.render(<TenDaysSelect/>, mountNode);
```
